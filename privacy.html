Privacy

No names or accounts are required.

Leave It Here is designed to be used without identity, profiles, or social pressure.

How this static demo works

Shared anonymous feelings can be stored in Supabase so visitors can see the same public room. Private releases, unsent letters, and time capsules stay in this browser's localStorage on this device.

Reports and comfort messages use a random browser token to limit repeated actions. Leave It Here does not require names, accounts, or profiles.

A future hosted version should keep the same principle: no names, no accounts, and no unnecessary identity data.
