Privacy

No names or accounts are required.

Open Quiet is designed to be used without identity, profiles, or social pressure.

How this static demo works

Because this version runs on GitHub Pages, it does not use a server database yet. Private releases, shared demo posts, reports, unsent letters, comfort events, and time capsules are stored in this browser's localStorage on this device.

That means shared posts are public only inside this local demo until a real database is connected. Letters and time capsules stay private to this browser. Clearing browser storage can remove them.

A future hosted version should keep the same principle: no names, no accounts, and no unnecessary identity data.
